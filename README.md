# List

Inspired by 'ls' unix command

Provides a better tree representation

### Requirements

- Nerdfont https://www.nerdfonts.com/font-downloads

### Install

#### Linux

```sh
cd <this folder>
sh ./install.sh
```

### Usage
``` bash
list <path> <flags separated by space>
```

#### Flags

- ```-R``` recursive 
- ```-d``` displays file size
- ```-a``` hidden files
- ```-V``` version
