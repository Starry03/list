### Linux

#### Install

```sh
cd <this folder>
sh ./linux_install.sh
```

#### Update

```sh
sh ./linux_update.sh
```

### Usage
``` bash
list <'path'> <'-flags'>
```

#### Flags

- ```R``` recursive 
- ```d``` file size
- ```a``` hidden files
- ```V``` version
- ```l``` permissions

### ls vs list

- list about 2x ls ram
- speed: really close to ls
